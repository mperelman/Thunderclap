# Temporary Files Folder

This folder is for ALL temporary files generated during testing, debugging, or development.

## What Goes Here

- Test outputs
- Query results saved to files
- Debug logs
- Temporary data files
- Scratch files
- Any file not needed permanently

## What Does NOT Go Here

- Production code
- Documentation
- Configuration files
- Data files needed by the system

## Cleanup

Files in this folder can be deleted anytime. This folder is in .gitignore.
